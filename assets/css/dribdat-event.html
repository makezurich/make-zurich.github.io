Check out the official guide to MakeZurich and find out how to connect the contents of the challenge boxes here: Getting Started guide ### Documenting your project Once you create a project using the form on this page, you can use Markdown formatting to write a **Description** of your project. After saving, you can **Edit** the project to change several customisation options, like an icon, color and image. You will also be able to update your progress level. Team members can sign in and use the **Join** button at the top of the project page once you have started it. Their public profile will then be linked to it, and they will also be able to make changes to the project data. It is also possible to publish your project in an open source community site. If you start a project on GitHub, Bitbucket or DokuWiki, you can quickly and easily sync its details here by entering an **Autofill link** and enabling "auto-update". You will then find a **Sync** button at the bottom of your project page for quickly pulling in changes. Need more help? Get in touch with everyone else involved on our Slack chat.
